# SpringBoard
스프링으로 게시판 만들기

### 개발환경

* Eclipse
* STS
* jdk 1.8
* tomcat 8.5
* Oracle SQL Developer / MyBatis
* JSTL / jQuery / bootstrap

### 링크

http://192.168.0.3:8080/board/list

### 기능
- Spring Web Security를 이용한 로그인 처리
  + 사용자 더미데이터 
    + 아이디 : user00~ user79, manager80~ manager80, admin90~admin99
    + 비밀번호 : pw00~pw99
  + 자동로그인 기능
- 게시글 CRUD 
- 페이징처리 / 검색처리
- Ajax를 이용한 댓글처리
- Ajax를 이용한 첨부파일 기능
